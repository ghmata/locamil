Informações sobre Exportação
Dados Exportados
  • Carros: Todos os veículos da frota (modelo, placa, cor, valor da diária)
  • Clientes: Todos os clientes cadastrados (nome, WhatsApp)
  • Locações: Todas as locações (ativas, finalizadas e canceladas)
Uso dos Arquivos
SQL: Execute os comandos INSERT em qualquer banco de dados SQL (MySQL, PostgreSQL, SQL Server) para recriar os dados.
CSV: Abra no Excel, Google Sheets ou importe em Python/Pandas para análise estatística e criação de gráficos.
JSON: Use em aplicações web, APIs ou scripts Python/JavaScript para processamento programático dos dados.
Atenção: Os arquivos exportados contêm dados sensíveis. Mantenha-os seguros e não compartilhe publicamente.
